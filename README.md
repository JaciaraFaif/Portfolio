# Portfolio
Este portfólio reúne os meus projetos desenvolvidos ao longo da minha formação em Engenharia da Computação, com foco em desenvolvimento web, programação e resolução de problemas computacionais.

Aqui apresento trabalhos práticos realizados com tecnologias como HTML, CSS, JavaScript, Java, C++ e MATLAB, demonstrando as minhas competências técnicas e evolução como estudante.

O objetivo deste portfólio é mostrar a minha capacidade de aplicar conhecimentos teóricos em projetos reais, bem como o meu interesse contínuo em aprender novas tecnologias e melhorar as minhas habilidades na área de desenvolvimento de software.
